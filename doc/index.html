README

TUgo is a tool to quickly query people, projects, courses or theses provided by the TISS API.

Requirements

This project requires Ruby with version 3.3.0, Rails with version 7.1.2 and SQLite3 with version 1.4.

Getting Started

To initialize this project, run the bundler to install the needed gems:

bundle install

To initialize the database, we use this command:

bin/rails db:migrate

After that, the server can be started via

bin/rails server

Documentation

TUgo also provides extensive YARD documentation, which can be generated if needed. Simply run the following command:

yardoc
